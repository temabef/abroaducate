## 📐 Usage Guidelines

### Minimum Sizes
- Full logo: 150px width minimum (optimized spacing)
- Icon version: 28px minimum (due to geometric detail)
- Compact version: 95px width minimum (optimized spacing)
- Social media "A" symbols: 64px minimum for profile pictures
- Favicon: 16px, 32px versions

### Spacing & Positioning
- Text positioned at optimal distance from "A" symbol for cohesion
- Clear space around logo should be at least 40% of the logo height
- "A" symbol and text form unified visual unit
- Social media versions are perfectly centered in 200x200px squares
- Never place text closer than the minimum clear space

### Do's
✅ Use on white or light backgrounds (main logo)  
✅ Use white version on dark backgrounds  
✅ Maintain aspect ratio when scaling  
✅ Use icon version for small spaces (navbar, favicon)  
✅ Ensure sufficient contrast with background  

### Don'ts  
❌ Don't modify colors unless using approved variants  
❌ Don't stretch or distort the logo  
❌ Don't place on busy backgrounds without contrast  
❌ Don't separate the "A" symbol from the text arbitrarily  
❌ Don't add additional colors or decorative elements  
❌ Don't use outdated complex or cluttered versions  

## 💻 Implementation

### Current Usage
- **Navbar**: `logo-icon.svg` (56x56px scaled to 32px)

- **Favicon**: `favicon-32x32.svg`, `favicon-16x16.svg`
- **Auth Error Page**: `logo-icon.svg` (48px)
- **Structured Data**: `logo.svg`
- **Marketing Materials**: `logo.svg` or `logo-white.svg`

### PWA & App Icons
- **iOS Home Screen**: `apple-touch-icon.svg` (180x180px)
- **PWA Installation**: `favicon-192x192.svg`, `favicon-512x512.svg`
- **Web App Manifest**: `manifest.json` with full icon set
- **Cross-Platform**: Complete favicon coverage for all devices

- **Profile Pictures**: `logo-a-background.svg` or `logo-a-white-background.svg`

- **App Icons**: `logo-a-square.svg` for transparent backgrounds
- **Platform Uploads**: Use any 200x200px "A" symbol variant based on platform theme
