# 💰 AcademyForge - Strategic Pricing Restructure

## 🎯 **CURRENT PRICING ANALYSIS**

### **Existing Structure Problems**
- **Free Plan:** 1 SOP, 3 AI improvements, 1 analytics, 1 plagiarism check
- **Basic Plan ($4.99):** 10 SOPs, 50 AI improvements, 10 analytics, 10 plagiarism checks  
- **Pro Plan ($19.99):** Unlimited everything

**Critical Issues:**
- ❌ **SOP-Only Focus:** Ignores 75% of our built features (Cover Letters, Personal Statements, Academic CVs)
- ❌ **Poor Free Tier:** Too restrictive to build engagement or showcase platform value
- ❌ **Undervalued Platform:** Pricing doesn't reflect comprehensive suite we've built
- ❌ **Limited Market:** Only targets graduate students (tiny market segment)

## 🔧 **TECHNICAL IMPLEMENTATION REQUIREMENTS**

### **Database Schema Updates**
```sql
-- New pricing plans table
CREATE TABLE pricing_plans (
    id VARCHAR PRIMARY KEY,
    name VARCHAR NOT NULL,
    monthly_price DECIMAL(10,2),
    annual_price DECIMAL(10,2),
    features JSONB NOT NULL,
    limits JSONB NOT NULL,
    stripe_price_id VARCHAR,
    is_active BOOLEAN DEFAULT true
);

-- Enhanced usage tracking
CREATE TABLE user_usage (
    user_id UUID REFERENCES users(id),
    feature_type VARCHAR NOT NULL,
    current_usage INTEGER DEFAULT 0,
    monthly_limit INTEGER,
    reset_date DATE,
    PRIMARY KEY (user_id, feature_type)
);

-- Feature access control
CREATE TABLE feature_gates (
    plan_id VARCHAR REFERENCES pricing_plans(id),
    feature_name VARCHAR NOT NULL,
    access_level VARCHAR NOT NULL, -- full, limited, none
    monthly_limit INTEGER, -- null = unlimited
    PRIMARY KEY (plan_id, feature_name)
);
```
